## рҹҡҖ Quick Start Guide

### Absolute Fastest Path to Launch

**Day 1: Setup (30 minutes)**

```bash
# 1. Install dependencies
npm install
cd studio && npm install && cd ..

# 2. Set up Sanity
cd studio
npx sanity init  # Follow prompts
npx sanity schema deploy
cd ..

# 3. Configure environment
# Copy .env.local.example to .env.local
# Add your Sanity project ID

# 4. Start servers
npm run dev  # Terminal 1
cd studio && npm run dev  # Terminal 2
```

**Day 2: Content (1 hour)**

- Visit http://localhost:3333 (Sanity Studio)
- Add Site Settings (about, email, social)
- Create Hero Section
- Add 1-3 Services
- Write 1-3 Blog Posts
- Add 1-2 Videos

**Day 3: Customize (30 minutes)**

- Update colors in `tailwind.config.js`
- Change company name in components
- Review all content

**Day 4: Deploy (20 minutes)**

- Push to GitHub
- Deploy to Vercel
- Deploy Sanity Studio
- Test live site

**Day 5: Launch! рҹҺү**

- Final checks
- Announce on social media
- Share with clients
